Cette extension web a été conçue pour renforcer la sécurité en ligne en identifiant les liens URL malveillants et homographiques. Son usage est accès sur la vérification des liens URL administratifs et gouvernementaux français.

Les sites de phishing utilisent de plus en plus des caractères cyrilliques pour imiter des noms de domaine légitimes. Ces attaques, connues sous le nom d’attaques homographiques, exploitent les similitudes visuelles entre certains caractères de différents alphabets pour tromper les utilisateurs.

Les liens dont le nom de domaine contient des caractères cyrilliques peuvent sembler authentiques visuellement, mais ils peuvent rediriger l'utilisateur vers une page de phishing. Il est crucial de vérifier l'authenticité des URL avant de cliquer, surtout lorsqu'il s'agit de sites administratifs, gouvernementaux, e-commerces, réseaux sociaux et postaux.

Voici deux liens visuellement identiques (homographiques) :

  • https://www.example.com
  • https://www.exаmple.com (contient un caractère "а" en cyrillique)

Les sites de phishing utilisant des caractères cyrilliques pour imiter des noms de domaine légitimes peuvent être hébergés sur des serveurs sécurisés et obtenir des certificats HTTPS.

Les certificats HTTPS sont délivrés par des autorités de certification (CA) qui vérifient la propriété du domaine, mais ils ne peuvent pas toujours détecter les noms de domaine trompeurs utilisant des caractères similaires. Par conséquent, même un site de phishing peut apparaître comme sécurisé avec un cadenas vert dans la barre d’adresse du navigateur.

Pour se protéger contre ces attaques, il est recommandé de :

  • Vérifier attentivement l’URL : Rechercher des caractères inhabituels ou des différences subtiles.
  • Utiliser des extensions de navigateur : Certaines extensions comme celle-ci peuvent aider à détecter les noms de domaine trompeurs.
  • Activer l’affichage Punycode : Certains navigateurs permettent d’afficher les noms de domaine en Punycode, ce qui rend les caractères non latins plus visibles.
Pour information :

Un nom de domaine est une adresse unique qui permet d'identifier un site web sur Internet.
Par exemple, dans l'adresse "www.example.com/accueil", "example.com" est le nom de domaine.

Vérifier un lien URL reçu par e-mail ou SMS.
Il est important de copier/coller le lien afin de détecter s'il ne s'agit pas d'un lien homographique (nom de domaine visuellement identique).
Cliquer droit sur le lien > Cliquer sur "Copier le lien"
